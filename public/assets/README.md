# Asset drop folder

Add generated PNG assets here with these exact names:

- `player-ship.png`
- `enemy-ship-1.png`
- `enemy-ship-2.png`
- `asteroid.png`
- `laser.png`
- `powerup.png`
- `background.png`

The game already has vector fallback art, so missing assets will not break gameplay.

Recommended style: premium mini arcade space game, clean neon sci-fi, readable at small size, dark space background, polished glow, not childish, not over-detailed, mobile game asset style.
